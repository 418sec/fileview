# Fileview
---

File browsers on web. It's easy to browser your local file.

---
## Installation

```
> npm install fileview -g
```

## Usage
```
> fileview -p /your file path/
```

## Help
```
> fileview -help
==========================================

Usage: fileview [options]

Options:
   -p,                 file path
   -P                  http port
   -h/-help/--help     output usage information

Examples:
$ fileview -p /projects/example/ -P 8080

==========================================
```
